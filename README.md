# Student Registration System

A console-based Java application developed using Java 23, JDBC, and MySQL for managing student records. This project allows users to add and view student information through a menu-driven interface.

## Features

- Add Student Records
- View Student Records
- MySQL Database Integration
- JDBC Connectivity
- Menu-Driven Console Application
- Object-Oriented Programming Concepts
- Prepared Statements for Secure Database Operations

## Technologies Used

- Java 23
- JDBC
- MySQL
- MySQL Connector/J
- NetBeans IDE

## Learning Outcomes

This project demonstrates:

- Java OOP Concepts
- JDBC API Usage
- MySQL Database Operations
- CRUD Operations
- Exception Handling
- Console-Based Application Development
